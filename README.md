# Employee Profile Generator

A lightweight Python script designed to automate the formatting of employee profiles and parse company asset codes. This project was built to practice and demonstrate foundational Python string manipulation skills.

## 🚀 Features
- **String Formatting:** Uses f-strings to create clean, readable employee cards.
- **Type Casting:** Smoothly integrates integers (age, salary) into string outputs.
- **Data Parsing:** Uses string slicing (`[start:end]`) to extract department codes, years, and initials from a standardized employee ID.

## 📦 Output Example
When you run the script, it outputs:
- A full text description of the employee.
- A cleanly formatted profile card.
- Separated data blocks (Department, Year, Initials) parsed from the employee code.

## 🛠️ Technologies Used
- Python 3
